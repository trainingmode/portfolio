My portfolio site is built with a custom static site generator.

Built on the Directory ‚Üí Article folder structure concept. Similar to other Static Site Generators like Jekyll or Gatsby.

For this project, I mixed pure HTML with Tailwind CSS (via CDN) for the template system. Generated sites are pretty lightweight, with only the Tailwind CDN dependency.

Internally, articles are converted using Pandoc. While Markdown is preferred, Pandoc will convert all files to HTML by default. You could write articles in any markup format that Pandoc supports.

I wrote a set of internal plugins to preprocess special Markdown syntax, like embedded iFrames and image comparisons.

I‚Äôve previously developed a full-fledged SSG, with a CommonMark parser from scratch, but I just needed a quick implementation to quickly get my site running.

What I‚Äôve learned from previous projects is to reduce or avoid complexity. I would have just written pure HTML, but that makes for a horrible experience writing articles.
